使用硬盘解锁实用工具为硬盘解锁

当您将设有密码保护的硬盘连接到没有安装 WD SmartWare™ 软件的计算机时，您可以使用硬盘解锁 实用工具为硬盘解锁。WD SmartWare 软件通过虚拟 CD (VCD) 盘提供硬盘解锁实用工具。打开 Windows 资源管理器时，该虚拟 CD 盘就显示在“我的电脑”下。

要为硬盘解锁：

  1. 通过下列一种方法在要解锁的硬盘上找到并打开 WD Unlocker 实用工具：
    • 使用计算机文件管理实用工具打开 WD Unlocker VCD 并双击“Unlock.exe”文件

    • 单击开始 > 我的电脑，并双击移动存储设备下的 WD Unlocker VCD 图标
  2. 在密码框中输入密码。
  3. 单击解锁为硬盘解锁。
  4. 在您的硬盘现已解锁提示框中，单击退出，关闭硬盘解锁实用工具。

说明：“密码无效”信息表示您输入了错误的密码。您可以再尝试四次。如果您连续五次输入无效密码，“密码尝试次数过多”对话框会通知您必须执行下列其中一项任务：

  • 重启硬盘并重试

  • 清空硬盘

重试：

  1. 安全断开硬盘与计算机的连接。
  2. 关闭硬盘电源，然后打开并重新连接到计算机。
  3. 重复上述解锁步骤。

如果忘记或丢失密码，WD 将无法为您找回密码，您必须清空硬盘后才能再使用它。因此，应写下密码并将它存放在远离计算机的安全地点，尤其是当您为多台设备设置不同密码保护时更是如此。
