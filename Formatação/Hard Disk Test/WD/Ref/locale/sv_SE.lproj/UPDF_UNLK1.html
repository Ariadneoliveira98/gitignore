Låsa upp disken med verktyget Lås upp disk

När du ansluter din lösenordsskyddade disk till en dator som inte har programvaran WD SmartWare™ installerad kan du använda verktyget Lås upp disk för att låsa upp disken där. Programvaran WD SmartWare innehåller verktyget Lås upp disk på en virtuell CD-enhet (VCD) som visas i Utforskaren under Den här datorn i Windows.

Låsa upp disken:

  1. Hitta och öppna verktyget WD Unlocker på den disk som du vill låsa upp genom att antingen:
    • Använd datorns filhanteringsverktyg för att öppna VCD:n WD Unlocker och dubbelklicka på filen Unlock.exe

    • Klicka på Start > Den här datorn och dubbelklicka på VCD-ikonen WD Unlocker under Enheter med flyttbara lagringsmedia
  2. Skriv ditt lösenord i rutan Lösenord.
  3. Klicka på Lås upp för att låsa upp din disk.
  4. Vid meddelandet Din disk är nu upplåst, klicka på Avsluta för att stänga verktyget Lås upp disk.

Obs! Ett meddelande om ogiltigt lösenord betyder att du skrev in fel lösenord. Du kan försöka igen fyra gånger till. Om du anger fem ogiltiga lösenord visas dialogrutan Alltför många lösenordsförsök som talar om att du måste antingen:

  • Starta om disken och försöka igen

  • Radera disken

Försök igen så här:

  1. Koppla säkert från disken från datorn.
  2. Slå av strömmen till datorn och sedan på den igen. Återanslut sedan strömmen till datorn.
  3. Upprepa upplåsningsproceduren.

Om du glömmer eller tappar bort ditt lösenord kan WD inte hämta det åt dig och du måste radera din disk innan du kan använda den igen. Av denna orsak bör du skriva ditt lösenord och spara det på ett säkert ställe borta från datorn, speciellt om du använder många lösenordsskyddade enheter och har flera lösenord.
