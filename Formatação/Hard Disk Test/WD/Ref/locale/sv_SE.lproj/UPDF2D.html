WD-disk används

Det går inte att uppdatera den inbyggda programvaran när My Book- eller My Passport-disken används.

Innan du kan uppdatera den inbyggda programvaran måste du avsluta programvaran WD SmartWare™ och stänga alla öppna program på datorn.

Fortsätt så här:

  1. Avsluta programvaran WD SmartWare.
  2. Stäng alla öppna program på datorn.
  3. Kör Uppdaterare för inbyggd programvara igen.
