Вход в систему с правами администратора

Для обновления микропрограммы накопителя My Book или My Passport необходимо войти в систему в качестве пользователя с правами администратора.

Порядок входа в систему с правами администратора:

  1. Нажмите Выход.

  2. Выполните выход из системы, а затем повторно выполните вход в качестве пользователя с правами администратора.

Примечание. Обычно, если вы владелец компьютера, по умолчанию вы являетесь администратором. Если компьютер в качестве администратора настраивал кто-либо другой — этот человек должен войти в систему как администратор и предоставить права администратора для вашей учетной записи пользователя.
