Entsperren Ihres Laufwerks mit dem Dienstprogramm Drive Unlock

Wenn Sie Ihr kennwortgeschütztes Laufwerk an einen Computer ohne installierte WD SmartWare™-Software anschließen, können Sie das Laufwerk mit dem Dienstprogramm Drive Unlock entsperren. Die WD SmartWare-Software enthält ein Dienstprogramm Drive Unlock auf einem virtuellen CD-Laufwerk (VCD-Laufwerk), das im Windows-Explorer unter "Arbeitsplatz" angezeigt wird.

So entsperren Sie das Laufwerk:

  1. Suchen und öffnen Sie das WD Unlocker-Dienstprogramm auf dem Laufwerk, das Sie entsperren möchten. Mögliche Methoden dazu sind:
    • Verwenden des Dateiverwaltungssystems Ihres Computers zum Öffnen der VCD "WD Unlock" und Doppelklicken auf die Datei "Unlock.exe"

    • Klicken auf Start > Arbeitsplatz und Doppelklicken auf das Symbol der VCD "WD Unlock" unter "Geräte mit Wechselmedien"
  2. Geben Sie Ihr Kennwort in das Feld Kennwort ein.
  3. Klicken Sie auf Entsperren, um Ihr Laufwerk zu entsperren.
  4. Bei der Eingabeaufforderung Ihr Laufwerk ist jetzt entsperrt klicken Sie auf Beenden, um den Bildschirm des Dienstprogramms Drive Unlock zu schließen.

Hinweis: Die Meldung "Ungültiges Kennwort" bedeutet, dass Sie das falsche Kennwort eingegeben haben. Sie haben vier weitere Versuche. Wenn Sie fünf ungültige Kennworte eingegeben haben, weist Sie das Dialogfeld "Zu viele Versuche bei der Kennworteingabe" darauf, dass Sie folgende Möglichkeiten haben:

  • Das Laufwerk neu starten und erneut versuchen

  • Das Laufwerk löschen

So versuchen Sie es erneut:

  1. Trennen Sie das Laufwerk sicher vom Computer.
  2. Schalten Sie das Laufwerk aus und wieder ein, bevor Sie es an den Computer anschließen.
  3. Wiederholen Sie das Entsperren.

WD kann Ihr Kennwort nicht für Sie abrufen, wenn Sie es vergessen oder verloren haben. In diesem Fall müssen Sie das Laufwerk erst löschen, bevor Sie es wieder verwenden können. Aus diesem Grunde ist es sinnvoll, das Kennwort zu notieren und an einem sicheren Ort abseits von Ihrem Computer aufzubewahren, insbesondere, wenn Sie mehrere Kennwort-geschützte Geräte haben und mehrere Kennworte verwenden.
