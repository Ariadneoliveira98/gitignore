Aktualisieren mehrerer Laufwerke

Sie können die Firmware nur auf jeweils einem Laufwerk gleichzeitig aktualisieren.

So aktualisieren Sie die Firmware auf mehreren Laufwerken:

  1. Trennen Sie alle My Book- und My Passport-Laufwerke außer dem einen, das Sie aktualisieren möchten.
  2. Führen Sie das Firmware-Aktualisierungsprogramm erneut aus.
  3. Trennen Sie das Laufwerk, das Sie gerade aktualisiert haben, und schließen Sie ein anderes an.
  4. Führen Sie das Firmware-Aktualisierungsprogramm erneut aus.
  5. Wiederholen Sie die Schritte 3 und 4, bis Sie alle Laufwerke aktualisiert haben.
