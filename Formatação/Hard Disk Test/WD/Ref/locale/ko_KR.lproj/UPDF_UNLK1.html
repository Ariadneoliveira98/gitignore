드라이브 잠금 유틸리티를 사용하여 드라이브 잠금 해제

암호로 보호된 드라이브를 WD SmartWare™ 소프트웨어가 설치되지 않은 컴퓨터에 연결할 때마다 드라이브 잠금 해제 유틸리티를 사용하여 드라이브의 잠금을 해제할 수 있습니다. WD SmartWare 소프트웨어는 Windows 탐색기의 내 컴퓨터 아래에 나타나는 Virtual CD 드라이브에 대한 드라이브 잠금 해제 유틸리티를 제공합니다.

드라이브 잠금을 해제하려면:

  1. 다음 중 하나를 사용하여 잠금을 해제할 드라이브에서 WD Unlocker 유틸리티를 찾아서 엽니다.
    • 컴퓨터의 파일 관리 유틸리티를 사용하여 WD Unlocker virtual CD를 열고 Unlock.exe 파일을 두 번 클릭합니다.

    • 시작 > 내 컴퓨터를 클릭하고 이동식 미디어 장치에서 WD Unlocker VCD 아이콘을 두 번 클릭합니다.
  2. 암호 상자에 암호를 입력합니다.
  3. 잠금 해제를 클릭하여 드라이브 잠금을 해제합니다.
  4. '드라이브가 잠금 해제되었습니다' 메시지에서 종료를 클릭하여 드라이브 잠금 해제 유틸리티를 닫습니다.

참고: 유효하지 않은 암호는 잘못된 암호를 입력했음을 나타냅니다. 암호 입력을 4회 다시 시도할 수 있습니다. 유효하지 않은 암호를 5회 입력하면 '암호 입력 시도 횟수를 초과했습니다' 대화 상자에 다음을 수행해야 한다는 메시지가 표시됩니다.

  • 드라이브 다시 시작 및 다시 시도

  • 드라이브 지우기

다시 시도하려면:

  1. 컴퓨터에서 드라이브를 안전하게 분리합니다.
  2. 드라이브의 전원을 껐다가 켠 다음, 컴퓨터에 다시 연결합니다.
  3. 잠금 해제 절차를 반복합니다.

암호를 잊어버렸거나 분실한 경우 WD가 사용자의 암호를 확인할 수 없으므로 드라이브를 다시 사용하기 위해서는 드라이브를 지워야 합니다. 이러한 이유로 암호를 기록하여 컴퓨터로부터 안전한 장소에 보관해두어야 하며, 많은 암호로 보호된 장치를 사용하고 암호가 여러 개인 경우 특히 권장됩니다.
