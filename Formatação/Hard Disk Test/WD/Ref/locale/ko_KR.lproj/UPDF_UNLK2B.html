드라이브 지우기

유효하지 않은 암호를 5회 입력하면 '암호 입력 시도 횟수를 초과했습니다' 대화 상자에 다음을 수행해야 한다는 메시지가 표시됩니다.

  • 드라이브 다시 시작 및 다시 시도

  • 드라이브 지우기

드라이브의 전원을 껐다가 다시 켜서 암호를 기억해내 다시 입력할 수 있지만, 암호를 잊어버렸거나 분실한 경우 WD가 사용자의 암호를 확인할 수 없으므로 드라이브를 다시 사용하기 위해서는 드라이브를 지워야 합니다. 이러한 이유로 암호를 기록하여 컴퓨터로부터 안전한 장소에 보관해두어야 하며, 많은 암호로 보호된 장치를 사용하고 암호가 여러 개인 경우 특히 권장됩니다.

드라이브를 지우려면:

  1. 암호를 묻는 메시지가 나타나지 않으면 다음 작업 중 하나를 수행하십시오.
    • 홈 탭의 '드라이브가 잠겨 있습니다' 메시지 위의 드라이브 이미지를 클릭하여 WD SmartWare 드라이브 잠금 해제 대화 상자를 표시합니다.

    • 다음 중 한 가지 방법으로 WD SmartWare 드라이브 잠금 해제 유틸리티를 시작합니다.
      • My Book/My Passport 드라이브의 하드웨어 구성에 따라 컴퓨터의 파일 관리 유틸리티를 사용하여 WD Unlocker VCD 또는 WD SmartWare VCD를 열고 Unlock.exe 파일을 두 번 클릭합니다.

      • 시작 > 내 컴퓨터를 클릭하고 WD Unlocker VCD 또는 이동식 미디어 장치 아래의 WD SmartWare VCD 아이콘.
  2. 다음과 같은 방법으로 드라이브 잠금을 해제하십시오(5회 시도 가능):
    1. 암호 상자에 암호를 입력합니다.
    2. 잠금 해제를 클릭합니다.
  3. 다섯 번째 입력한 암호도 틀리면 '암호 입력 시도 횟수를 초과했습니다'라는 메시지가 표시됩니다.
  4. 드라이브를 지울 경우 데이터 손실에 관한 경고를 읽어보십시오.
  5. 동의함 확인란을 클릭하여 위험 요소를 확인했음을 나타냅니다.
  6. 포맷을 클릭하여 해당 드라이브를 지웁니다.
