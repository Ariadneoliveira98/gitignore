Cancellazione dell'unità

Se s'inserisce per cinque volte la password sbagliata, una finestra di dialogo Numero eccessivo di tentativi di inserimento della password indica che occorre eseguire una delle due procedure seguenti:

  • Riavviare l'unità e riprovare

  • Cancellare l'unità

Benché sia possibile spegnere, quindi riaccendere l'unità e tentare di reinserire la password corretta, se la si è dimenticata o perduta, l'unità dovrà essere cancellata per poter essere nuovamente utilizzata. Per questo motivo, è utile trascrivere la propria password e conservarla in un luogo sicuro lontano dal computer, soprattutto se si utilizzano molti dispositivi ognuno dei quali è protetto da una password differente.

Per cancellare l'unità:

  1. Se non è visibile la richiesta di inserire la password, eseguire una delle operazioni successive:
    • Fare clic sull'immagine dell'unità sopra il messaggio L'unità è bloccata nella scheda Home per visualizzare la finestra di dialogo di sblocco dell'unità WD SmartWare.

    • Avviare l'utilità Sblocco dell'unità di WD SmartWare in uno dei seguenti modi:
      • Con l'utilità di gestione file del computer per aprire il VCD WD Unlocker o il VCD WD SmartWare, in base alla configurazione hardware dell'unità My Book o My Passport e fare doppio clic sul file Unlock.exe

      • Facendo clic su Start > Computer e doppio clic sul VCD WD Unlocker o il WD SmartWare VCD sotto Dispositivi con archivi rimovibili
  2. Per sbloccare l'unità, effettuare fino a cinque tentativi:
    1. Digitando una password nella casella Password.
    2. Facendo clic su Sblocca.
  3. Alla quinta password errata, viene visualizzata la finestra di dialogo Numero eccessivo di tentativi di inserimento della password.
  4. Leggere l'avviso circa la perdita dei dati in caso si cancelli l'unità.
  5. Fare clic sulla casella di controllo Accetto per indicare che si accetta il rischio.
  6. Fare clic su Formatta per cancellare l'unità.
