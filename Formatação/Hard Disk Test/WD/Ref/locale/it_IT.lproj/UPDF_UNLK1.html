Sblocco dell'unità mediante l'utilità Sblocco dell'unità

Ogni volta che l'unità protetta con password viene collegata a un computer su cui non è installato il software WD SmartWare™, è possibile utilizzare l'utilità Sblocco dell'unità per sbloccare l'unità. Il software WD SmartWare integra l'utilità Sblocco dell'unità su un'unità CD virtuale (VCD) visualizzata in Risorse del computer, nella schermata Esplora risorse.

Per sbloccare l'unità:

  1. Individuare e aprire l'utilità WD Unlocker sull'unità da sbloccare, in uno dei seguenti modi:
    • Utilizzando l'utilità di gestione file del computer per aprire il VCD WD Unlocker e facendo doppio clic sul file Unlock.exe

    • Facendo clic su Start > Risorse del computer e doppio clic sull'icona del VCD WD Unlocker, sotto Dispositivi con archivi rimovibili
  2. Digitare la password nella casella Password.
  3. Fare clic su Sblocca per sbloccare l'unità.
  4. Al prompt Unità sbloccata, fare clic su Esci per chiudere la schermata dell'utilità Sblocco dell'unità.

Nota: un messaggio Password non valida significa che la password immessa non è corretta. È possibile tentare altre quattro volte. Se s'inserisce per cinque volte la password sbagliata, una finestra di dialogo Numero eccessivo di tentativi di inserimento della password indica che occorre eseguire una delle due procedure seguenti:

  • Riavviare l'unità e riprovare

  • Cancellare l'unità

Per riprovare:

  1. Scollegare in sicurezza l'unità dal computer.
  2. Spegnere e riaccendere l'unità e ricollegarla al computer.
  3. Ripetere la procedura di sblocco.

Una password dimenticata o persa non viene automaticamente recuperata, pertanto l'unità dovrà essere cancellata per poter essere nuovamente utilizzata. Per questo motivo, è utile trascrivere la propria password e conservarla in un luogo sicuro lontano dal computer, soprattutto se si utilizzano molti dispositivi ognuno dei quali è protetto da una password differente.
