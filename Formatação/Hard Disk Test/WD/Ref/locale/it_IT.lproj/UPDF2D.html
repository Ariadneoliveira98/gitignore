L'unità WD è in uso

Il firmware non può essere aggiornato quando l'unità My Book o My Passport è in uso.

Prima di poter aggiornare il firmware, è necessario chiudere il programma WD SmartWare™ e tutte le applicazioni aperte sul computer.

Per continuare:

  1. Chiudere il software WD SmartWare.
  2. Chiudere le applicazioni aperte sul computer.
  3. Eseguire nuovamente il Programma di aggiornamento del firmware.
