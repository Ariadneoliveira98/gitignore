Déverrouiller votre disque dur à l'aide de l'utilitaire de Déverrouillage du lecteur

A chaque fois que vous connectez votre disque dur protégé par mot de passe à un ordinateur qui n'a pas le logiciel WD SmartWare™ installé, vous pouvez utiliser l'utilitaire Déverrouillage du lecteur pour déverrouiller le disque dur. Le logiciel WD SmartWare fourni l'utilitaire de déverrouillage du disque dur sur un CD virtuel (VCD) qui apparaît sous Poste de travail dans l'Explorateur Windows.

Pour déverrouiller le disque dur :

  1. Ouvrez l'utilitaire WD Unlocker sur le disque dur que vous voulez déverrouiller soit :
    • En utilisant le programme de gestion de fichier de votre ordinateur pour ouvrir le CD virtuel WD Unlocker et en double-cliquant sur le fichier Unlock.exe

    • En cliquant sur Démarrer > Poste de travail et en double-cliquant sur l'icône du CD virtuel WD Unlocker sous Lecteurs amovibles
  2. Tapez votre mot de passe dans la case Mot de passe.
  3. Cliquez sur Déverrouiller pour déverrouiller le disque dur.
  4. À l'invite Votre lecteur est maintenant déverrouillé, cliquez sur Quitter pour fermer l'écran de l'utilitaire de Déverrouillage du lecteur.

Remarque : Un message de mot de passe invalide indique que vous avez entré un mot de passe incorrect. Vous pouvez essayer encotre quatre fois. Si vous entrez cinq mots de passe incorrects, un message Trop de tentatives de saisie du mot de passe vous indique que vous pouvez :

  • Soit redémarrer le disque dur et réessayer

  • Ou Effacer le disque dur

Pour essayer à nouveau :

  1. Déconnectez en toute sécurité le disque dur de votre ordinateur.
  2. Faîtes redémarrer le disque dur, et reconnectez-le à votre ordinateur.
  3. Répétez la procédure de déverrouillage.

Si vous oubliez ou perdez votre mot de passe, WD ne pourra pas le récupérer pour vous et vous devrez alors effacer les données contenues sur votre lecteur avant de pouvoir le réutiliser. Pour cette raison, vous devez écrire votre mot de passe et de le conserver en lieu sûr loin de votre ordinateur, surtout si vous utilisez plusieurs périphériques protégés par mot de passe, avec des mots de passe différents.
