Effacement du disque dur

Si vous entrez cinq mots de passe incorrects, un message Trop de tentatives de saisie du mot de passe vous indique que vous pouvez :

  • Soit redémarrer le disque dur et réessayer

  • Effacement du lecteur

Parce que vous pouvez arrêter puis rallumer votre lecteur et tenter de vous souvenir de votre mot de passe pour le ressaisir en cas d'oubli ou de perte, WD ne peut pas le récupérer pour vous, et vous devez effacer les données présentes sur votre lecteur avant de le réutiliser. Pour cette raison, vous devez écrire votre mot de passe et de le conserver en lieu sûr loin de votre ordinateur, surtout si vous utilisez plusieurs périphériques protégés par mot de passe, avec des mots de passe différents.

Pour effacer le disque dur :

  1. Si aucun message n'apparaît pour vous inviter à fournir votre mot de passe, alors :
    • Cliquer sur l'image du disque au dessus du message Le lecteur est verrouillé sur l'écran d'accueil pour afficher l'utilitaire de déverrouillage du lecteur SmartWare.

    • Lancez l'utilitaire de déverrouillage du lecteur WD SmartWare soit en :
      • Utilisant le programme de gestion de fichier de votre ordinateur pour ouvrir le CD virtuel WDUnlocker ou le VCD WD SmartWare, selon la configuration matérielle de votre disque dur My Book ou My Passport et double-cliquez sur le fichier Unlock.exe.

      • En cliquant sur Démarrer > Poste de travail et en double-cliquant sur le VCD WD Unlocker ou sur l'icône WD SmartWare sous Disques amovibles.
  2. Faites cinq tentatives de déverrouillage du disque dur en :
    1. Tapant votre mot de passe dans la case Mot de passe.
    2. Puis en cliquant sur Déverrouiller.
  3. La cinquième tentative non valide fait apparaître le message Trop de tentatives de saisie du mot de passe.
  4. Lisez l'avertissement sur la perte de données en cas d'effacement du disque dur.
  5. Cliquez sur la case à cocher Je comprends pour indiquer que vous acceptez le risque.
  6. Cliquez sur Formater pour effacer votre disque dur.
