Le disque dur WD est en cours d'utilisation

Le micrologiciel ne peut pas être mis à jour quand le disque dur My Book ou My Passport est en cours d'utilisation.

Avant que le micrologiciel soit mis à jour avec succès, vous devez quitter WD SmartWare™ et fermer toutes les applications sur votre ordinateur.

Pour continuer :

  1. Quittez le logiciel WD SmartWare.
  2. Fermez toutes les applications sur votre ordinateur.
  3. Exécutez à nouveau le programme de mise à jour du micrologiciel WD.
